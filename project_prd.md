# Project Overview
- Name: ExamLens AI
- Founder: Abhijeet Kangane
- Description: A SaaS platform that analyzes historical exam papers (PDFs) using AI to map topic frequency, rank high-yield topics, and generate a smart study planner.
# Tech Stack
- Frontend: Next.js 14 (App Router), React, TypeScript, Tailwind CSS, Lucide React.
- Backend/Database: Supabase (Auth, Storage, PostgreSQL).
- AI Engine: Google Gemini 1.5 Flash API.
